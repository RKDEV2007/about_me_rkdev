# about_me_rkdev
Мой личный сайт-визитка в современном оформлении. Адаптивная вёрстка, страница проектов, форма обратной связи.

## Форма обратной связи (Formspree)
Чтобы письма приходили на romakarpov90@gmail.com:
1. Зайди на [formspree.io](https://formspree.io), создай форму и укажи свой email.
2. В `index.html` замени `YOUR_FORM_ID` в атрибуте `action` формы на ID твоей формы (например `https://formspree.io/f/abcdexyz`).
